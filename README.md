🏨 Hotel Bedroom Dashboard – Tableau Project

📌 Project Overview

This project presents an interactive Hotel Bedroom Dashboard built in Tableau. The dashboard analyzes hotel and Airbnb-style listing data to provide insights into room availability, pricing, capacity, and property details. It helps users understand booking trends and compare different listings effectively.

🎯 Objectives

- Analyze hotel/property listing information.
- Track room availability across different time periods.
- Compare prices, bedrooms, bathrooms, and accommodation capacity.
- Build an interactive dashboard for quick business insights.

🛠️ Tools Used

- Tableau
- Excel / CSV Dataset
- Data Visualization

📊 Dashboard Features

- Room Availability Analysis
- Price Comparison
- Bedrooms & Bathrooms Distribution
- Accommodation Capacity
- Interactive Filters
- Property Listing Overview

📈 Key Insights

- Identify highly available and frequently booked properties.
- Compare pricing across different listings.
- Understand the relationship between room size and accommodation capacity.
- Explore listing details using interactive filters.
